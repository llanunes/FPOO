## Operadores reelacionais [Comparação]

  * ">"    maior que
  * "<"    menor que
  * ">="   maior ou igual a
  * "<="   menor ou igual a
  * "=="   igualdade
  * "!="   desigualdade [diferença]

### Variaveis 
Espaço reservado [alocado] na memória RAM para guardar uma informação de um tipo pré-estabelecido. 


  Números inteiros:
  * byte    => 8 bits (vai do -128 ate o 127)
  * short   => 16 bits
  * int     => 32 bits
  * long    => 64 bits
